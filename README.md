# UI Sprint Design - projet THP 15 août 2018

## L'application

### Heroku

Vous trouverez l'application hébergée sur Heroku au lien suivant : https://ui-sprint-design-thp.herokuapp.com

### En local

Après avoir cloné le repo, run :

- `bundle install`

- `rails db:create`

- `rails db:migrate`
